Análisis de Datos mediante lenguaje R para la auditoría

El presente sitio web contiene información estadística extraída a partir de una base de datos. La información corresponde a los Estados de Situación Financiera y Estado de Resultados, los cuales se encuentran distribuidos de forma trimestral para los periodos 2010 hasta el primer trimestre del año 2023

Los paquetes utilizados fueron

  • tidyverse
  • stargazer
  • lubridate
  • forecast
  • readxl
